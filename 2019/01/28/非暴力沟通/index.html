非暴力沟通笔记

Posted on 2019-01-28

###有意识地使用语言

###非暴力沟通四要素

  • 观察
  • 感受
  • 需要
  • 请求

1、不要条件反射式地反应，而是去明了自己的观察、感受和愿望，有意识地使用语言

2、非暴力沟通的四个要素

  • 观察
  • 感受
  • 需求
  • 请求

3、非暴力沟通的过程

  • 什么是我的观察
  • 我的感受如何
  • 哪些需要（或价值、愿望等）导致那样的感受
  • 为了改善生活，我的请求是什么

4、区分观察和评论

  • 将观察和评论混为一谈，人们将倾向于听到批评，甚至会产生批判心理。
  • 非暴力沟通主张评论要基于特定时间和环境中的观察，而不鼓励绝对化的评论

5、区分想法和感受

  • 想法不带情绪，多使用【认为】这个词来表达想法
  • 感受带情绪，多使用【感到】这个词来表达感受
  • 语气、口吻、表情、行为动作都有可能在透露一个人的想法或者感受（情绪）

6、建立表达感受的词汇表

##7、情绪源于我们的需要是否得到满足

8、听到不中听的话时，我们可以有四种选择：

  • 责备自己
  • 责备他人
  • 体会自己的感受和需要
  • 体会他人的感受和需要

9、确认感受的方法

  • 尝试使用”我（感到）…因为我“
  • 刻意使用”对方（感到）…因为（对方）”

10、通过说出需求来向他人提出自己的主张，别人更容易接受，而不是通过批评

11、个人成长的三个阶段（还有后面的阶段）

  • 感情的奴隶——我们认为自己有义务使他人快乐
  • 面目可憎的时期——此时，我们拒绝考虑他人的感受和需要
  • 生活的主人——我们意识到，虽然我们对自己的 意愿、感受和行为负有完全的责任，但无法为他人负责。我们无法牺牲他人来满足自己的需要

12、关于请求

  • 表述越具体、清楚越好（一般而言）
  • 寻求反馈
  • 一旦人们认为不答应我们的请求会被惩罚，他们就会把我们的请求看做是命令。我们要清楚地表达我们无意强人所难。
